## "Ресурсы"
1. Здоровье экипажа
2. Ресурс корабля (хп корабля?)
3. Вода + еда
4. Кислород

События влияют на ресурсы. После каждого прожитого цикла некоторые ресурсы уменьшаются (тратятся). Для успешного завершения главы ресурсы должны находится в промежутках от n до m (точные значения будут определены потом)
## Работа
№ что (где) [ресурс]
1. Приготовление еды (кухня) [водаеда]
2. Вырвщивание овощей и фруктов (космо-ферма) [водаеда]
3. Фильтрация воды (блок фильтрации воды) [водаеда]
4. Восполнение кислорода (блок восполнения кислорода) [кислород]
5. Медобследование (медблок) [Здоровье]
6. Научная деятельность (лаборатория) [здоровье]/[хп корабля]/[водаеда]/[кислород]

Успешность и количество проделанной работы зависит от d20.
